# Python Data Analysis Project

This project demonstrates basic data analysis and visualization using Python.

## Features
- Loaded CSV data using Pandas
- Calculated average of selected columns
- Created visualizations using Matplotlib (bar chart, scatter plot, heatmap)

## Technologies Used
- Python
- Pandas
- Matplotlib

## Description
This project is part of the QSkill Internship beginner tasks. It focuses on understanding data manipulation and basic visualization techniques.
